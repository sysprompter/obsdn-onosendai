# Changelog

All notable changes to the Ono-Sendai theme are documented here.

## [1.0.0] — 2026-05-17

### Initial release

**Core palette — dark mode**
- Primary background `#17191a` (near-black, warm tint)
- Sidebar / panel background `#0f5172` (deep teal)
- Tab bar / status bar `#000000` (pure black)
- Primary text `#dcddde` (off-white)
- Accent `#28b0ff` (electric cyan-blue)
- H2 accent `#00ff37` (terminal green)
- Strong accent `#e60953` (hot magenta-red)
- Mild accent `#d39f2f` (warm amber)
- Scrollbar active thumb `#ff3392` (hot pink)

**Core palette — light mode**
- Primary background `#d1cabf` (warm beige)
- Sidebar background `#eeb157` (golden amber)
- Link / accent `#914a29` (rust/terracotta)
- H2 accent `#4d5c8b` (slate blue)

**Typography**
- Primary font: Cairo → Roboto → Inter → Segoe UI
- Monospace: Fira Code → Source Code Pro
- Body: 13 px / 1.65 line-height
- H1: 1.9 em with cyan bottom border
- H2: 1.6 em with charcoal bottom border
- H3–H6: cascading 10 px left-indent per level

**Layout**
- Content padding: 20 px top, 30 px right, 30 px bottom, 45 px left
- Square corners on all chrome elements; 4 px radius on content elements
- Active tab indicated by 2.5 px bottom accent stripe
- Resize handle: 4 px wide (vertical), 3 px tall (horizontal)
- File explorer: compact 12 px / 14 px, full filename wrap
- File explorer emoji icons — folder (📂/📁 collapsed), image (🖼️), canvas (📊), PDF (📄), vault root (💻)

**Components styled**
- Workspace layout, ribbon, titlebar
- Tab bar and tabs (dark + light)
- Left/right sidebars and leaf panels
- File explorer — folders, files, active state
- Tag pane panel text
- Reading view and Live Preview / source editor
- Headings (reading + CM6 editor)
- Body text, bold, italic, strikethrough
- Lists with indentation guide rails
- Inline code and fenced code blocks
- Syntax highlighting tokens
- External and internal links
- Tag pills with semantic colour variants
- Blockquotes
- Tables (header, alt rows, hover)
- Callouts
- Embeds / transclusions with whisper title
- Checkboxes with hue-rotate CRT filter
- Footnotes
- Images with hover zoom
- Search results and match highlighting
- Metadata / frontmatter panel
- Graph view colour assignments
- Status bar
- Modals and command palette
- Tooltips
- Scrollbars (10 px, square, hot-pink active)
- Settings panel
- Hover popover preview
